# 🔧 Sol Rosa Ajustes

Site institucional desenvolvido para a oficina **Sol Rosa Ajustes**, com foco em apresentar os serviços oferecidos, facilitar o contato com clientes e proporcionar uma experiência moderna e responsiva em qualquer dispositivo.

## ✨ Funcionalidades

- Página inicial com apresentação da empresa
- Design moderno e responsivo
- Botão para contato via WhatsApp
- Botão de localização com abertura no Google Maps
- Seção de serviços
- Informações de contato
- Layout otimizado para computadores, tablets e celulares
- Interface com gradientes e animações suaves

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts


O projeto foi desenvolvido para funcionar corretamente em:

- 💻 Desktop
- 📱 Smartphones
- 📟 Tablets


## 📈 Melhorias Futuras

- Formulário de contato
- Galeria de serviços realizados
- Sistema de orçamento online
- Área para avaliações de clientes
- Integração com Google Maps
